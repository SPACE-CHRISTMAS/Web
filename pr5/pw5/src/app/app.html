Про проєкт

Ця лабораторна робота реалізована на Angular з компонентним підходом, роботою з формами та інтеграцією з API для збереження списку завдань.

  • CRUD-операції для задач
  • Маркування задач за категоріями
  • Відмітка виконання та дедлайни

Технології

Angular TypeScript SCSS REST API
